説明書

  • このゲームについて
    • 2020/5/5(火)コミックマーケット98にて頒布予定だったゲームの体験版です
    • 本編全6ステージ(27マップ)+α予定のところ、3ステージ(11マップ)まで遊べます
    • Web公開版はサウンド周りの処置が一部(ループ処理等)ダウンロード版と異なります
  • 操作方法
    • C ... 決定 / ジャンプ
    • X ... キャンセル / 攻撃
    • Z ... サブウェポン使用 (HP回復)
    • スペース ... サブウェポン切り替え(体験版だと使う機会無い)
    • ESC ... ポーズ/メニュー開閉
    • ※ゲームパッドでも一応動くものの、まだあんまりテストしていないため非推奨
